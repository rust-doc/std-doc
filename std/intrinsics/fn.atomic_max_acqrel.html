使用有符号比较的当前值的最大值。

Maximum with the current value using a signed comparison.

此内在函数的稳定版本可通过“fetch_max”方法在“原子”有符号整数类型上使用，方法是将“Ordering::AcqRel”作为“order”传递。 例如，“AtomicI32::fetch_max”。

The stabilized version of this intrinsic is available on the atomic signed integer types via the fetch_max method by passing Ordering::AcqRel as the order. For example, AtomicI32::fetch_max.
