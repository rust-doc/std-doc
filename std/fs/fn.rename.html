Function std::fs::rename1.0.0[−][src]

pub fn rename<P: AsRef<Path>, Q: AsRef<Path>>(from: P, to: Q) -> Result<()>
Expand description

将文件或目录重命名为新名称，如果已存在则替换原始文件。

Rename a file or directory to a new name, replacing the original file if to already exists.

如果新名称位于不同的挂载点，这将不起作用。

This will not work if the new name is on a different mount point.

Platform-specific behavior

此函数当前对应于 Unix 上的 rename 函数和 Windows 上带有 MOVEFILE_REPLACE_EXISTING 标志的 MoveFileEx 函数。

This function currently corresponds to the rename function on Unix and the MoveFileEx function with the MOVEFILE_REPLACE_EXISTING flag on Windows.

因此，from 和 to 存在时的行为不同。 在 Unix 上，如果 from 是一个目录，to 也必须是一个（空）目录。 如果 from 不是目录，则 to 也必须不是目录。 相反，在 Windows 上，from 可以是任何东西，但 to 不能是目录。

Because of this, the behavior when both from and to exist differs. On Unix, if from is a directory, to must also be an (empty) directory. If from is not a directory, to must also be not a directory. In contrast, on Windows, from can be anything, but to must not be a directory.

请注意，这可能会在未来发生变化。

Note that, this may change in the future.

Errors

此函数会在以下情况下返回错误，但不仅限于这些情况：

This function will return an error in the following situations, but is not limited to just these cases:

  • 从不存在。
  • from does not exist.
  • 用户没有查看内容的权限。
  • The user lacks permissions to view contents.
  • from 和 to 位于不同的文件系统上。
  • from and to are on separate filesystems.

Examples

use std::fs;

fn main() -> std::io::Result<()> {
    fs::rename("a.txt", "b.txt")?; // Rename a.txt to b.txt
    Ok(())
}
