如果当前值与“旧”值相同，则存储一个值。

Stores a value if the current value is the same as the old value.

此内在函数的稳定版本可通过“compare_exchange”方法在“原子”类型上使用，方法是将“Ordering::SeqCst”作为“成功”和“失败”参数传递。 例如，“AtomicBool::compare_exchange”。

The stabilized version of this intrinsic is available on the atomic types via the compare_exchange method by passing Ordering::SeqCst as both the success and failure parameters. For example, AtomicBool::compare_exchange.
